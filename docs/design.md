# CyberPond 赛博鱼塘 — 设计规范

## 1. 色彩系统

### 1.1 主色板

| 色阶 | 色值 | 用途 |
|------|------|------|
| 极淡蓝 | `#E3F2FD` | 页面背景色 |
| 浅蓝 | `#BBDEFB` | 卡片/面板背景 |
| 主蓝色 | `#42A5F5` | 按钮、标题栏、强调元素 |
| 深蓝 | `#1E88E5` | 按钮按下态、重要文字 |
| 暗蓝 | `#1565C0` | 导航栏、状态栏 |

### 1.2 辅色

| 用途 | 色值 |
|------|------|
| 金币金色 | `#FFC107` |
| 成功/确认绿 | `#66BB6A` |
| 警告/提示橙 | `#FFA726` |
| 错误/删除红 | `#EF5350` |
| 白色文字 | `#FFFFFF` |
| 主文字色 | `#212121` |
| 次要文字色 | `#757575` |

## 2. 字体规范

| 用途 | 字号 | 字重 |
|------|------|------|
| 页面标题 | 24px | Bold |
| 板块标题 | 18px | Bold |
| 正文 | 14px | Regular |
| 辅助文字/标签 | 12px | Regular |
| 按钮文字 | 16px | Medium |
| 数值显示（金币等） | 20px | Bold |

### 2.1 图标风格
- 使用 Material Design 风格的线性图标
- 图标大小统一 24x24 像素（可缩放）
- 颜色跟随父级文字颜色

## 4. UI 布局规范

### 4.1 安全区域
- 内容距屏幕边缘 16px（左右边距）
- 顶部状态栏预留 24px
- 底部导航栏高度 56px

### 4.2 卡片规范
- 圆角半径：12px
- 阴影：轻微盒状阴影（Y 偏移 2px，模糊 8px，透明度 15%）
- 内边距：16px
- 卡片间距：12px

### 4.3 按钮规范
- 主按钮：填充主蓝色，白色文字，圆角 8px
- 次按钮：主蓝色描边，透明背景，主蓝色文字
- 高度：48px
- 最小宽度：120px
- 禁用态：灰色背景 `#BDBDBD`

### 4.4 底部导航栏
- 固定底部，56px 高度
- 白色背景，顶部 1px 分割线 `#E0E0E0`
- 3-4 个 tab 图标+文字
- 当前选中 tab 使用主蓝色
