# CyberPond 赛博鱼塘 — 开发执行步骤

> 每个 Phase 完成一个功能闭环，确认可运行后再进入下一 Phase。
> Phase 顺序设计原则：先核心数据后界面，先单功能后多系统联动。

## Phase 1：项目基础设施

**目标：** 搭建可运行的空壳项目，配置和存档读写就绪。

| # | 任务 | 产出 |
|---|------|------|
| 1.1 | 创建完整目录结构（已在 docs/technical.md 中定义） | 空文件夹就位 |
| 1.2 | 创建 JSON 配置文件（fish_types / pond_config / game_config） | 3 个 JSON 配置 |
| 1.3 | 创建 SaveManager.cs — 读写存档 JSON 到本地 | save/load 方法 |
| 1.4 | 创建 GameManager.cs — 游戏启动入口、初始状态加载 | AutoLoad 就位 |
| 1.5 | 创建 EconomyManager.cs — 金币增减接口 | add/spend 方法 |
| 1.6 | 创建 PlayerData.cs — 玩家数据模型 | 纯数据类 |

**验证标准：** 启动游戏，GameManager 加载配置和存档，日志输出当前金币数。

---

## Phase 2：鱼塘系统（核心）

**目标：** 能创建鱼塘、查看鱼塘详情。

| # | 任务 | 产出 |
|---|------|------|
| 2.1 | 创建 PondData.cs / FishData.cs 数据模型 | 数据类 |
| 2.2 | 创建 PondManager.cs — 鱼塘 CRUD 操作 | 增删改查方法 |
| 2.3 | 创建主界面场景 main_map.tscn（暂用占位背景替代地图） | 基础场景 |
| 2.4 | 创建鱼塘详情场景 pond_detail.tscn | 详情场景 |
| 2.5 | 创建 MainMapUI.cs — 显示鱼塘列表、跳转详情 | UI 交互 |
| 2.6 | 创建 PondDetailUI.cs — 展示鱼塘信息 | UI 交互 |
| 2.7 | 创建 CreatePondUI — 输入名称 + 保存鱼塘 | 创建流程 |
| 2.8 | 鱼塘解锁金币消耗逻辑 | 解锁检查 |

**验证标准：** 能创建鱼塘、切换多个鱼塘查看详情、解锁新鱼塘消耗金币。

---

## Phase 3：商店与背包

**目标：** 能购买鱼苗、出售鱼籽、查看背包。

| # | 任务 | 产出 |
|---|------|------|
| 3.1 | 创建 shop.tscn 商店场景 + ShopUI.cs | 商店界面 |
| 3.2 | 创建 inventory.tscn 背包场景 + InventoryUI.cs | 背包界面 |
| 3.3 | 鱼苗购买功能（金币扣除 → 加入背包） | 购买流程 |
| 3.4 | 鱼籽出售功能（背包扣除 → 金币增加） | 出售流程 |
| 3.5 | 底部导航栏 NavigationBar 切换场景 | 导航联动 |

**验证标准：** 商店买鱼苗→背包看到鱼苗，背包卖鱼籽→金币增加。

---

## Phase 4：养成系统

**目标：** 鱼苗投放到鱼塘，随时间成长，产出鱼籽。

| # | 任务 | 产出 |
|---|------|------|
| 4.1 | 创建 FishManager.cs — 鱼的状态管理与时间计算 | 成长逻辑 |
| 4.2 | 鱼苗投放功能（从背包选鱼苗→放入鱼塘） | 投放流程 |
| 4.3 | 成鱼状态显示（鱼塘详情中区分成长中/已成年） | 状态展示 |
| 4.4 | 鱼籽产出（成鱼每天自动积攒鱼籽） | 鱼籽累积 |
| 4.5 | 鱼籽收获功能（点击收获→鱼籽进背包） | 收获流程 |

**验证标准：** 买鱼苗→投放→等待（可通过改配置缩短测试）→成鱼→产籽→收获→出售，完整闭环跑通。

---

## Phase 5：岛屿与钓鱼系统

**目标：** 新增岛屿发现、船票商店、星露谷风格钓鱼小游戏。

| # | 任务 | 产出 |
|---|------|------|
| 5.1 | 创建 IslandManager.cs — 岛屿状态与船票消耗 | 岛屿管理 |
| 5.2 | 创建 islands.json — 4 个小岛配置 | 岛屿数据 |
| 5.3 | 创建 DiscoverUI + discover.tscn — 岛屿选择界面 | 发现页 |
| 5.4 | 底部导航改为 4 tab：鱼塘/发现/商店/背包 | 导航改造 |
| 5.5 | 商店增加船票购买、背包增加物品展示 | 商店/背包扩展 |
| 5.6 | 创建 FishingGame.cs — 星露谷风格钓鱼小游戏 | 钓鱼玩法 |
| 5.7 | 岛屿访问状态与离开确认逻辑 | 岛屿会话 |
| 5.8 | 钓鱼→捕获→背包/出售/放生完整流程 | 钓鱼闭环 |

**验证标准：** 商店买船票 → 发现页选择岛屿 → 消耗船票前往 → 点击水域钓鱼 → 成功捕获鱼进入背包 → 放入鱼塘养成。
