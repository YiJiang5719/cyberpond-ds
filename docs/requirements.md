# CyberPond 赛博鱼塘 — 开发需求文档

## 1. 项目概况

| 项 | 内容 |
|----|------|
| 游戏类型 | 2D 休闲养成手游 |
| 运行平台 | Android |
| 游戏引擎 | Godot 4.6（GL Compatibility 渲染器） |
| 开发语言 | C# (.NET) |
| 地图服务 | OpenStreetMap（免费，无 API Key 要求） |
| 网络需求 | 纯单机，数据存储于设备本地，地图加载需联网 |
| UI 风格 | 简洁扁平化，主色调淡蓝色（#E3F2FD / #90CAF9 / #42A5F5） |
| 时间系统 | 现实时间制（基于系统时间戳） |

## 2. 核心玩法循环

```
金币 → 购买鱼苗 → 投放至鱼塘 → 等待现实时间成长 → 成鱼每日产鱼籽
  ↑                                                      │
  └────────── 收获鱼籽出售给商店 ←──────────────────────┘
```

金币同时用于解锁更多鱼塘。

## 3. 功能模块详述

### 3.1 地图系统
- 集成 OpenStreetMap 瓦片地图显示
- 获取设备 GPS 定位，地图默认居中于当前定位
- 用户可拖拽/缩放地图浏览
- 鱼塘以图标标记显示在地图对应 GPS 坐标上
- 用户可在地图上点击位置创建新鱼塘

### 3.2 鱼塘系统
- 第一个鱼塘免费，额外鱼塘花金币解锁（见数值配置）
- 创建鱼塘：地图选点 → 输入自定义名称 → 确认创建
- 每个鱼塘绑定 GPS 坐标（lat / lon）
- 每个鱼塘有最大养鱼数量上限
- 鱼塘详情页展示：鱼塘名称、已养鱼列表、鱼的状态、可收获鱼籽数

### 3.3 商店系统
- 展示所有可购买的鱼苗种类及价格
- 拥有足够金币时可购买，购买后进入背包
- 可出售背包中的鱼籽换取金币
- 鱼苗买入价和鱼籽卖出价可通过 `shop_config.json` 的倍率统一调整

### 3.4 背包系统
- 背包用于存放鱼苗和鱼籽
- 初始 10 个格子，最大 30 个格子
- 每解锁一个额外格子花费 100 金币
- 每个格子存放一种物品，同种物品可堆叠
- 配置见 `configs/inventory_config.json`

### 3.5 养成系统
- 鱼苗从背包投放到指定鱼塘
- 投放后开始按现实时间计算成长
- 成长时间到达后变为成鱼状态
- 成鱼每天（24小时）自动产出鱼籽
- 鱼籽需手动收获，放入背包
- 鱼塘中有多条成鱼时，鱼籽量累积

### 3.6 经济系统
- 金币获取途径：
  - 在商店出售鱼籽
  - （未来可扩展：签到、任务等）
- 金币消耗途径：
  - 购买鱼苗
  - 解锁新鱼塘

### 3.7 存档系统
- 所有数据存储于本地（JSON 文件）
- 存档内容包括：
  - 玩家信息（金币数量）
  - 鱼塘列表（含名称、GPS 坐标、已养鱼）
  - 每条鱼的状态（投放时间、是否成年、累计未收鱼籽）
  - 背包内容
- 存档自动保存（关键操作后触发）

## 4. 数值配置总览

| 配置项 | 文件路径 |
|--------|----------|
| 鱼类数值 | `configs/fish_types.json` |
| 鱼塘解锁数值 | `configs/pond_config.json` |
| 商店价格倍数 | `configs/shop_config.json` |
| 背包格子配置 | `configs/inventory_config.json` |
| 游戏全局参数 | `configs/game_config.json` |

## 5. 界面清单

| 界面名称 | 功能说明 |
|----------|----------|
| 主地图界面 | 地图展示 + GPS 定位 + 鱼塘标记 + 底部导航栏 |
| 鱼塘详情页 | 查看鱼塘内鱼类状态、收获鱼籽、投放鱼苗 |
| 商店界面 | 浏览鱼苗列表、购买、出售鱼籽 |
| 背包界面 | 查看持有的鱼苗和鱼籽 |
| 创建鱼塘页 | 地图选点 + 输入名称 + 确认创建 |
